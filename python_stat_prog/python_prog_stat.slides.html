import numpy as np

def ecdf(data):
    """Calcula la funcin de distribucin acumulada emprica 
    para un arreglo de medidas de una sola dimensin."""

    # Cantidad de observaciones: n
    n = len(data)

    # Datos en el eje horizontal: x
    x = np.sort(data)

    # Datos en el eje vertical: y
    y = np.arange(1, len(data)+1) / n

    return x, y

Inferencia estadística, ¿qué es?¶

  • La inferencia estadstica consiste en el conjunto de mtodos y procedimientos mediantes los cuales se determinan propiedades de una poblacin basndose en datos muestrales, aplicando para ello el razonamiento inductivo (de lo particular a lo general) as como mediciones y controles del posible error asociado a los datos y proceso.

  • Entre dichos mtodos y procedimientos se encuentran: el muestreo, la estimacin (frecuentista vs bayesiana, paramtrica vs no-paramtrica), la prueba o contraste de hiptesis, el diseo de experimentos.

Inferencia estadística: ¿es necesaria en la llamada era del "Big Data"?¶

Si leemos algunas de los enunciados de la llamada era del Big Data, parecería que la inferencia estadística es cosa del pasado...¶

... Sin embargo, aún en la llamada era del Big Data, la inferencia estadística es necesaria por varias razones:¶

  1. Casi siempre lo que tenemos es "small data".
  2. Aun en el caso de tener un gran volumen de datos, el aprendizaje automtico (machine learning) necesita de la inferencia, entre otras cosas: para asegurar que las informaciones arrojen la suficiente variedad y veracidad, para entrenar, validar y calibrar ciertos modelos y algoritmos, para establecer la ocurrencia de patrones, etc.
  3. El investigador no tiene ni procesa Big Data en su cabeza, pero s muchas hiptesis o planteamientos parciales.
  4. Incluso la inteligencia artificial tendr que lidiar con tcnicas de muestreo, experimentacin y quiz hasta partir de sus propias premisas y hiptesis.

Prueba de hipótesis (1/2)¶

  • En estadstica, una hiptesis es una afirmacin sobre un parmetro de la poblacin.

  • La hiptesis nula es una afirmacin que no se rechaza a menos que los datos de la muestra evidencien que es falsa.

  • Es posible que la diferencia observada entre la media de los hombres y la de las mujeres sea algo aleatorio o un simple error.

  • Hiptesis nula $H_0: \mu_F = \mu_M$. Es decir, no existe diferencia entre la media salarial de hombres y de mujeres de la DGII.

  • Y, cmo evaluamos esta hiptesis?

  • Una prueba de hiptesis estadstica, con respecto a alguna caracterstica desconocida de la poblacin de inters, es cualquier regla para decidir si se rechaza la hiptesis nula con base a una muestra aleatoria de la poblacin.

Prueba de hipótesis (2/2)¶

  • Casi nunca tenemos acceso directo a los datos de la poblacin, de manera que tenemos que recurrir a algo que la represente, al menos en sus parmetros de inters. Para ello recurrimos a lo que se denomina un estadstico de prueba.

  • Asimismo necesitamos un criterio que nos ayude a validar nuestra hiptesis a travs de dicho estadstico de prueba.

  • Como queremos evitar equivocarnos, partimos de la probabilidad de rechazar la hiptesis nula condicionada a que la misma sea cierta:

    $$P(\text{rechazar}\hspace{0.1cm} H_0 \hspace{0.1cm}\mid \hspace{0.1cm} H_0 \hspace{0.1cm}\text{es}\hspace{0.1cm}\text{cierta}) = \alpha$$

  • Bajo este contexto, el valor $p$ $p$-valor se define como la probabilidad de obtener un estadstico de prueba que sea igual o ms extremo que lo observado, asumiendo que la hiptesis que se est probando sea cierta. Con este criterio, y el nivel de confianza preestablecido para la prueba, podemos evaluar que tan "pequea" debe ser dicha probabilidad para poder rechazar la hiptesis nula sin riesgo de equivocarnos.

  • Resaltamos que el $p$-valor no nos dice per se que la hiptesis nula sea verdadera o falsa, sino tan solo que, condicionado a que la misma sea cierta y tomando como base el estadstico de prueba, la probabilidad de obtener datos muestrales que evidencien que nos equivocamos al rechazar dicha hiptesis no supera el porcentaje del $p$-valor.

Métodos de resampling (1/2)¶

  • En principio, para obtener un estadstico que represente la poblacin y realizar nuestras pruebas, necesitaramos entre otras cosas: conocer el proceso de generacin de datos en dicha poblacin (-> Distribuciones de Probabilidad), verificar si se cumplen las condiciones de los parmetros para cada tipo de distribucin (ejemplo: normalidad, independencia, ...), etc.

  • Por ahora, a falta de este conocimiento, vamos a recurrir a mtodos de resampling (remuestreo) para "simular" las caractersticas de dicha poblacin a partir de los datos disponibles.

  • Entender la inferencia estadstica a partir de los mtodos de resampling tiene numerosas ventajas:

    1- Aprendemos directamente de los datos, apoyndonos en la variabilidad presente entre las muestras de un mismo dataset, ms que en las asunciones paramtricas sobre una poblacin quiz desconocida o inaccesible. Casi como construir una poblacin simulada o sinttica a partir de los mismos datos disponibles; con la posibilidad de mejorar nuestros planteamientos a medida que incorporamos nueva informacin.

    2- Con esto nos acercamos al enfoque del machine learning (datos -> algoritmos -> conocimiento), sin menoscabo del requerido dominio sobre el tema de investigacin y el planteamiento legtimo de hiptesis.

    3- Asimismo, muchos de los mtodos de resampling (bootstrapping, cross-validation, etc.) son inherentes al proceso de machine learning, y forma parte esencial de ciertas familias de algoritmos.

    4- En general y hasta cierto punto, entender los mtodos de resampling nos lleva al corazn mismo de la inferencia estadstica. Ya que son mtodos a partir de los cuales podemos crear nuestros propios procedimientos y algoritmos.

Métodos de resampling (2/2)¶

Prueba de hipótesis mediante permutación¶

Pasos:¶

1- Calculamos la media para cada grupo. Despus calculamos la diferencia entre las medias, $D_\text{obs}$.


2- Partiendo de la hiptesis que no existen diferencias entre los dos grupos, reasignamos, o lo que es lo mismo, intercambiamos las etiquetas entre las observaciones de ambos grupos, y obtenemos as todos los posibles arreglos de reasignacin (mtodo exacto) o al menos una buena aproximacin de los mismos.


3- Para cada arreglo o permutacin calculamos a $D$, la diferencia entre las dos medias. Nota: Cada diferencia de medias (o de otro estadstico de prueba que usemos) para cada arreglo muestral se denomina rplica.


4- Encontramos el p-valor.

$$p-\text{valor}_\text{superior} = \frac{\text{nmero de}\hspace{0.1cm} \text{D's} >= D_\text{obs}}{\text{nmero de rplicas}}$$

$$p-\text{valor}_\text{inferior} = \frac{\text{nmero de}\hspace{0.1cm} \text{D's} <= D_\text{obs}}{\text{nmero de rplicas}}$$

$$p-\text{valor}_\text{2-colas} = \frac{\text{nmero de}\hspace{0.1cm} \mid\text{D's}\mid \hspace{0.1cm} >= \hspace{0.1cm}\mid D_\text{obs}\mid}{\text{nmero de rplicas}}$$

5- Comparamos el $p$-valor con el nivel de significancia preestablecido y llegamos a una conclusin.

def permutation_sample(data1, data2):
    """Genera una muestra de permutacin a partir de dos conjuntos de datos."""

    # Concatenamos los datos
    data = np.concatenate((data1, data2))

    # Permutamos los arreglos de datos concatenados
    permuted_data = np.random.permutation(data)

    # Separamos los arreglos permutados en dos
    perm_sample_1 = permuted_data[:len(data1)]
    perm_sample_2 = permuted_data[len(data1):]

    return perm_sample_1, perm_sample_2

# Una rplica de permutacin consiste en el valor de un estadstico calculado a partir de una muestra de permutacin

def draw_perm_reps(data_1, data_2, func, size=1):
    """Genera mltiples rplicas de permutacin."""

    # Inicializa un arreglo de rplicas
    perm_replicates = np.empty(size)

    for i in range(size):
        # Genera una muestra de permutacin
        perm_sample_1, perm_sample_2 = permutation_sample(data_1, data_2)

        # Calcula el estadstico de prueba
        perm_replicates[i] = func(perm_sample_1, perm_sample_2)

    return perm_replicates

def diff_of_means(data_1, data_2):
    """Diferencia entre las medias de dos arreglos (arrays)."""

    # Diferencia entre las medias de data_1, data_2
    diff = np.mean(data_1) - np.mean(data_2)

    return diff

# Calculamos la diferencia de medias entre el salario de los hombres y el de las mujeres
empirical_diff_means = diff_of_means(feb_sal_f, feb_sal_m)

# Generamos 10,000 rplicas de permutacin
perm_replicates = draw_perm_reps(feb_sal_f, feb_sal_m,
                                 diff_of_means, size = 10000)

# Calculamos el p-valor
p = np.sum(np.abs(perm_replicates) >= np.abs(empirical_diff_means)) / len(perm_replicates)

# Imprimimos los resultados en la consola

_ = sns.distplot(perm_replicates, hist=True)

# Intervalo de confianza de las diferencias mediante bootstrap

# Concatenamos los salarios
sal_concat = np.concatenate((feb_sal_f, feb_sal_m))

# Inicializamos un arreglo de rplicas de bootstrap
bs_replicates = np.empty(10000)

for i in range(10000):
    # Generamos la muestra de bootstrap
    bs_sample = np.random.choice(sal_concat, size=len(sal_concat))

    # Calculamos las rplicas
    bs_replicates[i] = diff_of_means(bs_sample[:len(feb_sal_m)],
                                    bs_sample[len(feb_sal_m):])
    
# Calculamos un intervalo de confianza de un 97.2%
conf_int = np.percentile(bs_replicates, [1.4, 98.6])

# Imprimos en la consola
print('intervalo de confianza 97.2% =', conf_int, 'Diferencia Salarial RD$')
